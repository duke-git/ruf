#### Ruf是一个Rust语言工具函数库。它提供了许多工具函数，让你的rust开发变得更简单.

_Ruf目前正处于开发中，你可能需要等到v1.0发布时再将其应用到你的生产之前。_

_Ruf是”rust util function"的缩写。_

## [English](./README.md)

## 特性

- 💪 强大: 支持常用Rust开发特性, string, collection, net, crypto, concurrency, etc。
- 💎 纯净: 保持最小范围外部依赖。
- 🛠 简洁: 结构组织良好，测试所有API函数。
